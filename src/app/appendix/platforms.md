# 平台功能

ClassIsland 支持多个平台，其部分功能在这些平台上的可用性存在差异。本文将介绍其平台功能在各个平台上的可用性情况。

## 图例

- ✅: 功能完全可用
- ⚠️: 功能在此平台上部分实现了
- ❌: 功能理论上在此平台可实现，但暂未实现
- ⛔: 功能在此平台上不可实现

## 功能可用性表

| 功能/平台 | Windows | Linux | macOS |
| --- | --- | --- | --- |
| 基础功能 | ✅ | ✅ | ✅ |
| RawInput | ✅ | ⛔ | ⛔ |
| 窗口置顶 | ✅[^1] | ✅ | ✅ |
| 窗口平台功能（如前台窗口判断等）| ✅ | ✅ | ⚠️[^2] |
| 插件 | ✅ | ✅[^3] |  ❌[^4] |
| 系统 TTS[^5] | ✅ | ❌ | ❌ |
| 定位 | ✅ | ❌ | ✅ |
| 自动更新 | ✅[^6] | ✅[^6] | ❌ |
| 应用内设置自动启动 | ✅[^7] | ✅[^7] | ❌[^8] |
| 注册 Url 协议 | ✅ | ❌ | ❌ |
| 窗口背景材质 | ✅[^9] | ⛔ | ⚠️[^10] |

[^1]: 在 Windows 中存在窗口段的设计，ClassIsland 在未取得 UIAccess 令牌时，无法将窗口覆盖到部分具有更高窗口段的系统置顶窗口（如“开始”菜单、任务管理器、通知中心等）
[^2]: 受 MacOS 平台限制，只实现了部分窗口属性的读取，并且只能在切换应用时触发前台窗口更新事件。
[^3]: 在部分国产 Linux 发行版上（如统信 UOS），在没有启用开发者模式的情况下，插件在加载本机库时可能会被系统拦截。
[^4]: 受 .NET MacOS 平台特性影响，暂时不支持加载插件。我们会尝试在未来解决这个问题。
[^5]: EdgeTTS、GPT-Sovits 等非系统 TTS 语音引擎不受影响。
[^6]: 仅在便携版可用。
[^7]: 可能会被安全软件拦截。在安全软件提示是否允许 ClassIsland 自动启动或类似提示时，请点击允许。
[^8]: 需要在系统设置中手动为 ClassIsland 启用自启动。
[^9]: AcrylicBlur 效果仅在 Windows 10.0.18362.0 及以上版本可用，Mica 效果仅在 Windows 10.0.22000.0 及以上版本可用。
[^10]: 仅 AcrylicBlur 效果可用。
